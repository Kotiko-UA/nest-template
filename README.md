## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Migration

```bash
# add new
$ npm run migration:generate:dev db/migrations/name-of-migration

```

## Endpoint

```bash
# add new
$ nest g resource name-of-endpoint --no-spec

```
